# is-car-service-station
Цей рядок додано для тестування Pull Request.

Цей репозиторій створений для лабораторної роботи №5 з дисципліни **Управління IT-проектами**.

## Опис проекту
"Інформаційна система контролю виконання робіт на СТО" — це система, що дозволяє:
 - відстежувати виконання робіт по автомобілях;
 - контролювати процеси технічного обслуговування;
 - автоматизувати комунікацію між майстрами та клієнтами;
 - зберігати стан виконання робіт у цифровому форматі.
